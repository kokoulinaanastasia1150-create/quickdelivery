## Введение

Огонёк — это интерактивная веб-платформа для заказа товаров с доставкой. 

Назначение продукта: Обеспечить удобный и быстрый способ заказа товаров повседневного спроса с доставкой на дом.

## Примеры использования

Ниже представлен список ключевых функций платформы, разделенный по ролям пользователей.

### Для покупателя:

1.  Регистрация/Вход
2.  Поиск товара
3.  Фильтрация каталога
4.  Добавление в корзину
5.  Изменение количества
6.  Выбор способа оплаты
7.  Указание времени доставки
8.  Подтверждение заказа

### Для курьера:

1.  Просмотр новых заказов
2. Принятие заказа в работу
3. Изменение статуса доставки
4. Визуализация интерфейса

## Технические требования

Для корректной работы платформы Огонёк необходимо соблюдение следующих условий:

*   Операционная система
*   Браузер
*   Интернет-соединение
*   Разрешение экрана
